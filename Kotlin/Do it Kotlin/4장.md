# `조건문`

```kotlin
fun main() {
    val a = 12
    val b = 7
    
    val max = if (a > b) {
        a  // max 에 a 할당
    } else {
        b  // max에 b 할당
    }
}
```

## `in 연산자와 범위 연산자로 조건식 간략하게 만들기`

```kotlin
else if (score >= 80.0 && score <= 89.9)
else if (score in 80.0 ... 89.9)
```

즉, `변수 이름 in 시작 값 ... 마지막 값`을 사용하여 간략하게 만들 수 있습니다.

## `when문으로 다양한 조건 처리하기`

```kotlin
when (x) {
    1 -> print("x == 1")
    2 -> print("x == 2")
    else -> {
        print("x는 1, 2가 아닙니다")
    }
}

when (x) {
    1, 2 -> print("x == 1")
    else -> {
        print("x는 1, 2가 아닙니다")
    }
}

when (x) {
    in 1..10 -> print("x는 1 이상 10 이하 입니다.")
    !in 10..20 -> print("x는 10이상 20 이하 입니다.")
    else -> print()
}
```

## `반복문`

```kotlin
for (x in 1..5) { // 1~5 출력
    print(x) // "12345"
}

for (x in 5..1) { 
    print(x) // 아무것도 출력되지 않음
}

for (x in 5 downTo 1) {
    print(x)  // "54321" 역순으로 출력
}

for (x in 1..5 step 2) {
    print(x)  // "135"
}

for (x in 5 downTo 1 step 2) {
    print(x)  // "531"
}
```
