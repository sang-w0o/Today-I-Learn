## `UTC 이후 밀리초`

```
1646376185642
```

`JWT`를 사용할 때 만료시간을 보거나 어떤 시간 값을 보면 위와 같이 알아보기 힘든 값으로 나와 있을 때가 있다. 

저게 어떤 값인지 매번 그냥 넘어갈 때가 있었는데 저 값이 어떤 값인지 알게 되어서 정리한다.

```
UTC: 1970년 1월 1일 자정
```

즉, `UTC` 이후부터 현재시간 까지 밀리초의 차이를 구한 것이다.

`밀리초 -> 년`으로 변환하면 위와 같이 52년 XX일 정도 되는 것을 확인할 수 있다.
